# C Programlama - Algoritma ve Programlama II Çalışma Soruları

Bu depo, **Konya Teknik Üniversitesi Bilgisayar Mühendisliği** bölümü bünyesinde aldığım Algoritma ve Programlama II dersine ait çalışma sorularını, sınav hazırlık notlarını ve uygulama çözümlerini içermektedir.

##  İçerik
- **Dinamik Bellek Yönetimi:** `malloc`, `calloc`, `realloc`, `free` kullanımı.
- **Veri Yapıları:** Struct (Yapılar) ve Union kullanımı.
- **Dosya İşlemleri:** C dilinde dosya okuma ve yazma pratikleri.
- **Gelişmiş Algoritmalar:** Rekürsif fonksiyonlar ve karmaşık döngü yapıları.
